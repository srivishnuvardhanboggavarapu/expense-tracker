# Personal Expense Tracker

## Project Overview

This project helps users track their daily expenses.
Users can add, view, search, update, and delete expenses.
The system generates reports and summaries using SQLite database.

---

## Features

- Add Expenses
- View Expenses
- Search Expenses
- Search by Date Range
- Update Expenses
- Delete Expenses
- Monthly Summary
- Average Expense
- Category Report
- Top Spending Category

---

## Technologies Used

- Python
- SQLite
- SQL
- VS Code
- GitHub

---

## Database Tables

### users
Stores user information.

### categories
Stores expense categories.

### expenses
Stores expense details.
